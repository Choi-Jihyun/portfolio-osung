오성노트 바로가기
기획 의도
과학을 공부하는 학생들이 헷갈릴 수 있는 주제에 대해 이해를 돕고자 '오성노트'를 제작했습니다. 현재는 척추 동물의 구분에 대한 노트가 있습니다. 척추 동물의 구분은 학생이 어려움을 겪는 주제 중 하나입니다. 이 사이트가 이러한 어려움을 해소하고, 어려운 주제에 대한 명확한 개념을 가질 수 있도록 돕고자 제작하였습니다.
Notion API
Notion API를 사용했습니다. 노션의 데이터 베이스를 사용하여 내용을 추가하고 삭제시에 더 편하게 할 수 있습니다.
filter를 사용한 분류
map과 filter를 사용하여 사용자가 버튼을 누르면 해당 태그의 게시물만 보여질 수 있도록 했습니다.
깔끔한 코딩
코딩 유지보수를 용이하게 하기 위하여 코딩의 가독성을 높이기 위해 노력했습니다.
