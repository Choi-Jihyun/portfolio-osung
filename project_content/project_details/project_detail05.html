웹 사이트 바로가기
기획 의도
사용자들의 운동 루틴을 공유하는 사이트가 있으면 좋겠다는 생각에 제작하게 되었습니다.
반응형, 적응형
모바일 전용 웹앱이기는 하지만 다양한 사용자들이 이용할 것을 고려하여 반응형도 신경을 썼습니다.
Firebase Realtime Database
firebase Realtime Database와 storage를 활용하여 사용자들이 실시간으로 자신의 운동 루틴을 공유할 수 있도록 했습니다.
Firebase Authentication
Firebase Authentication 사용하여, 사용자들의 이름을 가지고 운동 루틴을 공유할 수 있도록 하였습니다.
