웹 사이트 바로가기
React
CSR와 SPA(Single Page Application)를 구현하였습니다. 서버 측에서 렌더링하는 대신 브라우저에서 렌더링하는 방식입니다. 필요한 데이터만 불러와 화면을 갱신하는 방식으로 작동합니다. 이를 위해 React Router를 사용하였습니다.
그 외에 다음과 같은 React의 훅들을 적용하였습니다.
useState, useEffect, useRef, useParams, useContext, useNavigate, immer, Custom Hook 등.
Firebase Authentification
useAuthContext() 라는 firebase에서 제공하는 로그인 훅을 이용하였습니다. authContextProvider를 앱을 감싸게 하여 로그인을 했을 때 페이지가 다르게 보일 수 있도록 했습니다. 이를 활용하여 관리자인지 일반 사용자인지 구분하여 UI가 각 이용자에 맞추어 바꾸었습니다. 모바일에서는 메뉴를 열었을 때, 사용자의 이름을 포함하여 인사하는 기능을 추가하였습니다.
Firebase Realtime DataBase(1)
Gallery에서 useParams 훅을 이용하여 메뉴에서 바로 들어올 수 있도록 하였습니다. 메뉴에서 createSearchParams를 이용하여 던진 값을 해당 카테고리로 갈 수 있도록 URLSearchParams를 사용하여 받는 방식입니다. 해당 갤러리는 인증받은 관리자만 Firebase Realtime DataBase에 데이터를 올릴 수 있도록 하였습니다.
Firebase Realtime DataBase(2)
관리자가 데이터 베이스에 게시물을 올릴 수 있는 기능을 추가하였습니다. 날짜는 현재 날짜를 불러올 수 있도록 하고, 활동 사진을 여러 장을 선택하여 저장할 수 있도록 하였습니다. 저장한 사진은 Firebase에 storage로 저장한 후에 그 저장 이름을 불러와 gallery에 게시하도록 하였습니다.
차별화
swiper custom 하여 안정적이면서도 more beautiful 합니다. 카카오맵 API 추가하여 프로젝트에 연동했습니다. 또한, Json파일 이용하여 Json 파일에서 내용을 가져와 사이트를 구성하였습니다. 이후에 수정사항이 생길 때, Json 코드 하나만 수정하게 하여 유지보수가 용이합니다. 상단 메뉴도 다른 웹 사이트들과 다르게 기능적으로도 시각적으로도 gorgeous 합니다.
