메인 페이지
로고는 버려지는 것들을 재활용한다는 의미를 가진 브랜드의 아이덴티티를 살리기위한 메인 로고 애니메이션으로 배치하였습니다. 이때 marine serre 로고가 regenerated 라는 단어로 바뀌는 과정에서 겹치는 알파벳은 재구성되고 없는 알파벳만 추가되어 새로운 단어를 제시하여 글씨를 '재'활용합니다.
협업
git과 GitHub로 협업 작업을 진행하였습니다. 특정 부분을 작업하고 merge 하면서 발생한 충돌 문제와 같이, 다양한 시행착오가 있었습니다. 이를 통해서 어떻게 협업하는 것이 좋을지, 어떻게 효율적인 의사소통을 할 수 있는지 배웠습니다.
사용자 경험(UX)
위 페이지는 제가 주도적으로 코드를 작성한 페이지 중 하나입니다. 버튼을 눌렀을 때, 왼쪽 오른쪽 화살표가 비활성화될 수 있도록 하여 사용자 경험을 높였습니다.
Ecofuturism
위 페이지는 제가 주도적으로 코드를 작성한 인터렉션 중 하나입니다. 에코퓨처리즘은 에코와 퓨처리스트 합성어로 마린세르 브랜드가 가장 중요하게 생각하는 신념입니다. 이와 관련된 다양한 가치들이 스크롤을 하면 바다 속으로 빠져들어, 해당 브랜드가 자연 친화적임을 강조하였습니다.
의류 코어라인
위 페이지는 협업을 통해 코드를 작성한 인터렉션 중 하나입니다. 사진이 움직이면서 진열된 느낌과 쇼핑몰 느낌의 갤러리 효과를 주었습니다.
