기획 의도
제가 만든 웹 사이트들을 보여주기 위해 웹 사이트를 제작하였습니다.
반응형, 적응형
반응형을 사용하여 데스크탑에서 모바일까지 쉽게 볼 수 있도록 제작하였습니다.
SEO 최적화(1)
SEO 최적화를 위해 제가 만든 다른 웹사이트도 적용한 메타 태그 최적화뿐만 아니라 다양한 방법을 사용하여 SEO 최적화에 신경을 썼습니다. HTTPS를 사용하여 웹사이트를 안전하게 유지하여 보안 문제를 강화하였습니다. 또한, XML 사이트맵을 생성하고 Google Search Console에 제출하여 검색 엔진이 웹사이트의 페이지를 쉽게 색인화할 수 있도록 하였습니다.
SEO 최적화(2)
구글 검색시 제 포트폴리오 사이트가 최상단에 위치하는데 성공하였습니다!
Accessibility
접근성을 높이기 위해 많은 노력을 했습니다. 먼저, axios를 사용하여 사용자에 따라서 필요한 정보만 불러들일 수 있도록 하여, 초기 로딩 웹 페이지 로딩 속도를 높였습니다. 또한, png 파일을 webp 파일 형식으로 바꾸어 사이트의 속도를 높였습니다.
